f(x + ^x) - f(x)
--------------- = f ' (x) при малых ^x. Почему? Потому что f ' (x) это функция, которая показывает скорость изменения процесса (функции), т.е. описанное отношение в левой части.
        ^x

следовательно, f(x + ^x) - f(x) = f ' (x) * ^x => ^y = f ' (x) * ^x и вот это приближенное значение приращения функции называется ДИФФЕРЕНЦИАЛОМ функции и обозначается как dy:
dy =  f ' (x) * ^x.

чем меньше ^x - тем меньше погрешность. Потому что тем ближе наше отношение к производной.

и еще один вывод, f(x + ^x) - f(x) = f ' (x) * ^x => f(x + ^x)  = f(x) + f ' (x) * ^x = f(x) + f ' (x) * dx
